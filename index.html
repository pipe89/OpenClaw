OpenClaw

OpenClaw is a command-line application that allows a user to sign in with their Google account and use Google APIs from a local CLI tool.

What OpenClaw does

OpenClaw helps a user authenticate with Google and perform actions permitted by the Google APIs that the user has chosen to authorize.

The application is intended for use from the command line and is not a public consumer website.

How it works

When a user signs in, OpenClaw requests access only to the Google account permissions needed for the features the user chooses to use. After authorization, the application stores authentication tokens locally on the user's machine so the user can continue using the CLI without signing in every time.
